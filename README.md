# Super Mario Lite Lite

A vanilla JavaScript platformer game inspired by classic Super Mario Bros.

## Play Online

Visit the [GitHub Pages deployment](https://yourusername.github.io/super-mario-lite-lite/) to play the game.

## Run Locally

No build tools or dependencies required! Simply:

1. Clone this repository
2. Open `index.html` in your web browser

That's it! The game will run directly in your browser.

## Features

- 2 levels with unique themes (Overworld and Underground)
- Classic platformer mechanics: jumping, running, enemy stomping
- Coins and question blocks
- Particle effects and animations
- Mobile touch controls support
- Responsive design

## Controls

- **Desktop**: Arrow keys or WASD to move, Space/Up arrow to jump
- **Mobile**: Touch controls appear automatically

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Under "Source", select the `main` branch
4. Click Save

Your game will be live at `https://yourusername.github.io/repository-name/`

## Files

- `index.html` - Game structure and UI
- `game.js` - All game logic and rendering
- `styles.css` - CSS styles
